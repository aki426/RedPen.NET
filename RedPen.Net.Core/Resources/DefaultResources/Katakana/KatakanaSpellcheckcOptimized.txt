# 最適化の1つとして、間違いやすいオーストリアとオーストラリアを両方とも辞書から外した。
# 同様に、混同しやすいカタカナ語は辞書から外した。
# 旧JIS規格には準拠しない方針で、一般的に末尾に長音記号が必要とされるカタカナ語は長音記号を補った。
# 表記ゆれが起きやすいカタカナ語は、インターネット検索で一般的な用例を採用し、どちらともとれる場合はユーザがValidatorの結果から選択できるようにするためどちらも辞書から外した。
# 外国の名前に関しては、特に表記ゆれが発生しやすいものは辞書から外した。そもそもすべて辞書から外すべき、または人名辞書として分割するなどの議論は必要。
# お笑い芸人の名前が散見されるがそもそも辞書から外すべきかどうか議論が必要。
# 企業名や商品名が散見されるがそもそも辞書から外すべきかどうか議論が必要。
# サムネイル、サムネール両方登録があった場合は、より一般的な方、特に「イ」による表現が不自然でない場合は「イ」を優先した。
